What are SVGs?

Scalable Vector Graphics (SVG) are an XML-based markup language for describing two-dimensional based vector graphics. ... SVG is, essentially, to graphics what HTML is to text. SVG images and their related behaviors are defined in XML text files, which means they can be searched, indexed, scripted, and compressed. One of the pros of SVG is that the quality is not lost. Meaning, unlike images, you can make the picture as big as you want but it will not loose its image quality. One of the downside is that SVG cannot make images look realistic and thus have to like more or less cartoony. However, all of the component designs are 2d, thus the code in CursorBridge is optimized for SVG.
